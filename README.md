# ☁️ Microsoft AZ-104: Azure Administrator Labs

Welcome to my Azure Administrator portfolio! This repository contains my hands-on Infrastructure as Code (IaC) templates and automation scripts created while preparing for the **Microsoft AZ-104** exam. 

### 🛠️ Technologies Used:
* **PowerShell & Azure CLI:** For resource management and automation.
* **ARM Templates & Bicep:** For automated, declarative infrastructure deployment.
* **Microsoft Entra ID (Graph API):** Identity and access management.

### 📁 Repository Structure:
* **[01_Entra_ID_Management](./01_Entra_ID_Management)**: Scripts for managing users, groups, and B2B guests.
* **[02_Virtual_Networks](./02_Virtual_Networks)**: VNet peering, NSG rules, and Bastion setups.
* **[03_Compute_and_VMs](./03_Compute_and_VMs)**: Automated VM deployments and disk management.

*(I am actively updating this repository as I progress through my 30-day study plan!)*
